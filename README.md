# 欢迎使用 COC 名词翻译器 COC_Translater
##项目背景

COC名词一大堆查起来很麻烦就写了一个小python
代码很烂但是可以用

##安装

点击clone or download，点击Download ZIP，点击下载。
下载之后解压压缩包即可

##使用

将Translationtable.txt与COC.exe放在同一目录下
win环境下打开COC.exe文件
待排序完成后将需要翻译的文本复制粘贴到程序内回车即可翻译
